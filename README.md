# 科学计算器

这是一个用 Python 编写的科学计算器

## 主要功能

### 科学计算器功能

-   基本运算：加(+)、减(-)、乘(×)、除(÷)
-   科学计算：幂运算(^)、平方根(√)
-   三角函数：sin、cos、tan
-   对数运算：log
-   阶乘计算：n!
-   错误处理：友好的错误提示
-   自适应界面：自动调整字体大小

### 彩蛋功能

在计算器输入框中输入"tetris"

## 快速开始

1. 确保安装 Python 3.6+和必要的库：

    ```bash
    pip install pygame
    ```

2. 运行计算器：

    ```bash
    python main.py
    ```
